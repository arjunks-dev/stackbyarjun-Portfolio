### Project Overview

As part of the organization's digital transformation initiative, I successfully planned and executed the migration of the enterprise network security infrastructure from FortiGate firewalls to the Check Point Security and Cloud SD-WAN platform across multiple resort properties. The objective was to modernize network security, improve WAN utilization, enhance visibility, and provide secure connectivity for both on-premises and remote users.

The project involved designing and implementing a centralized security architecture while ensuring uninterrupted business operations for hospitality systems, corporate services, guest networks, and cloud-hosted applications.
